# live.rbg.tum streamcapture
Capture livestreams from TUM and store them locally

## Dependencies
This tool uses bash and Python 3.10 or later.
### Python dependencies
- Selenium 4.3.0
- youtube-dl 2021.12.17

### Other dependencies
- Firefox
- geckodriver
